## Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR-USERNAME/orof-pages.git
cd orof-pages
```

### 2. Add Visual Assets

Download your images from the Gemini shares and place them in the `images/` folder:
- `background-pattern.webp`
- `hero-optimal-failure.webp`
- `icon-optimal-failure.webp`

See [SETUP.md](SETUP.md) for detailed instructions.

### 3. Configure Google Form

Follow the [Google Form Integration Guide](SETUP.md#google-form--sheets-integration) to set up form submissions.

### 4. Test Locally

```bash
# Using Python
python -m http.server 8000

# Or using Node.js
npx http-server -p 8000

# Then visit: http://localhost:8000
```

### 5. Deploy to GitHub Pages

```bash
git add .
git commit -m "Initial deployment"
git push -u origin YOUR-BRANCH-NAME
```

Enable GitHub Pages in your repository settings.
